### 🫀 Cuffless Blood Pressure Measurement

> Using **ECG + PPG signals** and **deep learning** to estimate blood pressure without a traditional cuff.

**Key components:**
- Hardware: `AD8232` (ECG) + `MAX30102` (PPG) + `ESP32`
- Dataset: **MIMIC-III** (PhysioNet)
- Models: `BiLSTM` / `CNN-LSTM`
- Feature extraction: **PTT / PAT** (Pulse Transit Time)
- Validation: Bland-Altman analysis
